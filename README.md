# рҹ“– Deskripsi Proyek

Veloura Shoes merupakan website e-commerce yang dibangun menggunakan HTML5, CSS3, dan JavaScript (Vanilla JS). Website ini dirancang untuk memberikan pengalaman berbelanja online yang mudah, cepat, dan nyaman bagi pengguna.

Website ini juga mengimplementasikan konsep dasar e-commerce, user interface (UI), user experience (UX), manajemen produk, local storage, dan dokumentasi bisnis sebagai bagian dari tugas akhir mata kuliah Business Planning.
